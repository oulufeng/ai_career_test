# 职业功能测试系统 - 快速使用指南

## 🎯 方式一：本地直接运行（推荐新手）

### 步骤 1：安装 Node.js
1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本（推荐 v18 或更高）
3. 安装完成后重启电脑

### 步骤 2：安装依赖
```bash
# 打开命令提示符（CMD 或 PowerShell）
cd C:\Users\28226\Desktop\职业功能测试
npm install
```

### 步骤 3：启动服务器
```bash
npm start
```

### 步骤 4：访问应用
打开浏览器访问：http://localhost:3000

## 🔧 常见问题

### Q: npm 命令不存在？
A: 需要先安装 Node.js，下载地址：https://nodejs.org/

### Q: 端口 3000 被占用？
A: 修改 .env 文件中的 PORT 为其他端口，如 PORT=3001

### Q: AI 分析失败？
A: 检查 DeepSeek API Key 是否正确，或是否有足够的 API 余额

### Q: 如何在云服务器上部署？
A: 参考 README.md 中的"云服务器部署"章节

## 📞 技术支持

如遇问题，请检查：
1. Node.js 版本 >= 18.0.0
2. 端口 3000 是否被占用
3. .env 文件配置是否正确
4. 网络连接是否正常
